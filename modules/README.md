# Python Modules

## What is a Module?

A module is a Python file that contains reusable code such as functions, variables, and classes.

Modules help us organize large programs into smaller, manageable files.

---

## mymodule.py

This file is a custom module created by us.

It contains reusable functions:

- `greet()` → Returns a greeting message
- `add_numbers()` → Adds two numbers
- `multiply_numbers()` → Multiplies two numbers

---

## main.py

This is the main program file.

It imports the custom module using:

```python
import mymodule
